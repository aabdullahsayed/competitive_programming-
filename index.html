Competitive Programming Roadmap

LinkedIn Profile

From Noob to Grandmaster

  1. Beginner Topics
    • Introduction to Algorithms and Data Structures

      Learn the fundamentals of algorithms and data structures.

      Suggested time: 2 weeks

      Resources:

      • GeeksforGeeks - Data Structures
      • GeeksforGeeks - Algorithms
    • Basic Programming Techniques: Recursion, Sorting

      Understand basic programming techniques and concepts like recursion and sorting.

      Suggested time: 1 week

      Resources:

      • GeeksforGeeks - Recursion
      • GeeksforGeeks - Sorting Algorithms
    • Basic Math for Competitive Programming

      Learn essential mathematical concepts relevant to competitive programming.

      Suggested time: 1 week

      Resources:

      • TopCoder - Mathematics for Topcoders
      • CP-Algorithms - Competitive Programming Algorithms
  2. Intermediate Topics
    • Dynamic Programming

      Learn about dynamic programming techniques and problem-solving strategies.

      Suggested time: 3 weeks

      Resources:

      • GeeksforGeeks - Dynamic Programming
      • TopCoder - Dynamic Programming
    • Graph Theory: Basics and Algorithms

      Understand graph theory concepts and algorithms such as BFS, DFS, and shortest paths.

      Suggested time: 2 weeks

      Resources:

      • GeeksforGeeks - Graph Algorithms
      • CP-Algorithms - Graph Theory
    • Advanced Data Structures: Segment Trees, Fenwick Trees

      Explore advanced data structures for efficient query and update operations.

      Suggested time: 2 weeks

      Resources:

      • GeeksforGeeks - Segment Trees
      • GeeksforGeeks - Fenwick Trees
  3. Advanced Topics
    • Advanced Graph Algorithms: Minimum Spanning Tree, Floyd-Warshall

      Understand advanced graph algorithms and their applications.

      Suggested time: 2 weeks

      Resources:

      • GeeksforGeeks - Minimum Spanning Tree
      • GeeksforGeeks - Floyd-Warshall Algorithm
    • Advanced Dynamic Programming: Bitmasking, Digit DP

      Explore advanced techniques in dynamic programming for optimization problems.

      Suggested time: 2 weeks

      Resources:

      • HackerEarth - Bitmasking
      • GeeksforGeeks - Digit DP
    • Advanced Number Theory: Sieve Methods, Modular Arithmetic

      Learn advanced number theory concepts crucial for competitive programming.

      Suggested time: 2 weeks

      Resources:

      • CP-Algorithms - Sieve of Eratosthenes
      • GeeksforGeeks - Modular Arithmetic
  4. Expert Topics
    • Competitive Programming Contests and Strategies

      Learn about participating in competitive programming contests and strategies to excel.

      Suggested time: 4 weeks

      Resources:

      • TopCoder - Competitive Programming
      • Codeforces - Contests
    • Problem-Solving Techniques: Greedy, Divide and Conquer

      Master problem-solving techniques such as greedy algorithms and divide-and-conquer.

      Suggested time: 2 weeks

      Resources:

      • GeeksforGeeks - Greedy Algorithms
      • GeeksforGeeks - Divide and Conquer
    • Advanced Data Structures: Trie, Suffix Arrays

      Explore specialized data structures like trie and suffix arrays for advanced problem-solving.

      Suggested time: 2 weeks

      Resources:

      • GeeksforGeeks - Trie
      • GeeksforGeeks - Suffix Arrays

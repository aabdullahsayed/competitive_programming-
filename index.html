Roadmap from Noob to Grandmaster

  1. Stage 1: Beginner

    • Introduction to Programming

      Learn the basics of programming including syntax, variables, and control structures.

      Suggested time: 2 weeks

      Resources:

      • Learn C
      • Learn C++
    • Basic Data Structures: Arrays, Strings

      Understand the fundamental data structures and their operations.

      Suggested time: 1 week

      Resources:

      • GeeksforGeeks - Arrays
      • GeeksforGeeks - Strings
    • Basic Algorithms: Sorting, Searching

      Learn how to implement and use basic sorting and searching algorithms.

      Suggested time: 1 week

      Resources:

      • GeeksforGeeks - Sorting Algorithms
      • GeeksforGeeks - Searching Algorithms
    • Solving Easy Problems on Platforms like Codeforces, CodeChef

      Start solving basic problems to get familiar with competitive programming platforms.

      Suggested time: 2 weeks

      Resources:

      • Codeforces
      • CodeChef

  2. Stage 2: Intermediate

    • Advanced Data Structures: Stacks, Queues, Linked Lists

      Learn about advanced data structures and their applications.

      Suggested time: 2 weeks

      Resources:

      • GeeksforGeeks - Stacks
      • GeeksforGeeks - Queues
      • GeeksforGeeks - Linked Lists
    • Graph Algorithms: DFS, BFS, Dijkstra's Algorithm

      Understand graph traversal algorithms and shortest path algorithms.

      Suggested time: 2 weeks

      Resources:

      • GeeksforGeeks - DFS
      • GeeksforGeeks - BFS
      • GeeksforGeeks - Dijkstra's Algorithm

    • Stage 3: Advanced

      • Dynamic Programming

        Learn about dynamic programming techniques and problem-solving strategies.

        Suggested time: 3 weeks

        Resources:

        • GeeksforGeeks - Dynamic Programming
        • TopCoder - Dynamic Programming
      • Advanced Graph Algorithms: Minimum Spanning Tree, Floyd-Warshall

        Understand advanced graph algorithms and their applications.

        Suggested time: 2 weeks

        Resources:

        • GeeksforGeeks - Minimum Spanning Tree
        • GeeksforGeeks - Floyd-Warshall Algorithm
